ModMedic v1.0.0
================

Plugin error diagnostics for Paper servers.

=== Quick Start ===

1. Copy "server\ModMedic-1.0.0.jar" to your Paper server's plugins/ folder
2. Restart the server
3. Double-click "desktop\bin\ModMedicDesktop.bat" to launch the GUI

The plugin auto-connects to the desktop app at ws://localhost:9876.
